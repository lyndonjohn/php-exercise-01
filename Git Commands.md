1. Clone remote project to your local machine (you must be in your htdocs directory before cloning)
    -> git clone https://github.com/lyndonjohn/php-exercise-01.git

2. Add new files to commit (use only if you have a newly created file)
    -> git add .

3. Commit changes (its like saving your work before you publish it online)
    -> git commit -m 'insert meaningful description here'

4. Upload changes to remote repo
    -> git push origin master

5. Download changes from remote repo
    -> git pull --all
